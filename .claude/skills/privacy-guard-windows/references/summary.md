# Windows 隐私卫士 — 摘要

## 目的
检测 Windows 设备上可能存在的监控软件、间谍程序和隐私侵害行为，保护用户个人隐私。

## 检测范围
- **进程检测**：已知商业间谍软件、无签名后台进程、隐藏窗口进程、可疑路径进程
- **持久化检测**：注册表启动项（12 个关键路径）、Windows 服务、计划任务、WMI 事件订阅、Startup 文件夹
- **网络检测**：异常端口、外部连接、DNS 配置、系统代理/PAC、防火墙规则、网络嗅探驱动
- **文件系统检测**：已知监控软件安装路径、隐藏可执行文件、最近安装的程序、DLL 注入、IFEO 劫持
- **证书检测**：非标准根证书（系统级+用户级）、HTTPS 中间人攻击检测
- **策略检测**：组策略审计、远程桌面状态、RDP 登录记录、网络共享、剪贴板同步
- **高级检测**：AppInit_DLLs 注入、WMI 持久化、Image File Execution Options 劫持

## 核心产出
- 综合一键扫描脚本（PowerShell）
- 结构化风险报告
- 处置指南和加固建议
