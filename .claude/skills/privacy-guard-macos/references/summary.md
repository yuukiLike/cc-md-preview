# macOS 隐私卫士 — 摘要

## 目的
检测 macOS 设备上可能存在的监控软件、间谍程序和隐私侵害行为，保护用户个人隐私。

## 检测范围
- **进程检测**：已知商业间谍软件、键盘记录器、远程控制工具、隐藏进程
- **持久化检测**：LaunchAgents/Daemons、登录项、Cron、内核扩展、MDM 配置描述文件
- **网络检测**：异常端口、外部连接、DNS 篡改、代理设置、VPN/隧道、混杂模式嗅探
- **文件系统检测**：已知监控软件安装路径、隐藏目录、最近修改的可疑文件
- **证书检测**：非标准根证书、HTTPS 中间人攻击检测
- **权限检测**：辅助功能、屏幕录制、摄像头、麦克风、完全磁盘访问权限审计
- **浏览器检测**：Chrome/Safari/Firefox 扩展审查

## 核心产出
- 综合一键扫描脚本（Bash）
- 结构化风险报告
- 处置指南和加固建议
